## Penjelasan
Program ini adalah simulasi sederhana Manajemen Perpustakaan Manga berbasis Java. Fitur utama program meliputi CRUD (Create, Read, Update, Delete) untuk Buku Manga. Program menggunakan ArrayList untuk menyimpan Buku Manga, serta percabangan dan perulangan agar menu dapat dipilih dan dijalankan berulang hingga user memilih keluar. Program ini terinspirasi dari konsep seperti sistem peminjaman buku di perputakaan umum, tetapi ini hanya ada menu manga dan banyak jenis cerita nya masing-masing.

## Susunan Penjelasan Alur Program
1. Program menampilkan menu utama yang berisi: Tambah Manga, Lihat Semua Manga, Update Manga, Hapus Manga, dan juga Keluar.

2. Nah, disini pengguna bisa memilih salah 1 dari 5 opsi, yang dimana masing-masing mempunyai fungsi berbeda.

3. Jika pengguna memilih bagian "Tambah Manga" maka pengguna akan di minta untuk mengisi terlebih dahulu keterangan Manga tersebut, meliputi Judul Manga, Author, Genre, dan juga Tahun Rilis yang akan di simpan dalam ArrayList.

4. Pada opsi "Lihat Manga" akan menampilkan output list Manga yang sudah di tambah, tetapi jika belum menambah Manga, maka akan menampilkan output "Belum ada manga yang terdaftar".

5. Selanjutnya di opsi "Update Manga" akan melakukan perubahan keterangan Manga yang sudah di tambahkan sebelumnya, yaitu meliputi seperti judul, author, genre, dan juga tahun rilisnya.

6. Lalu pada opsi "Hapus Manga" tentunya akan menghapus Manga yang sudah di tambahkan sebelumnya.

7. Dan yang adalah menu akan terus tertampil secara berulang jika pengguna tidak memilihh opsi "Keluar" yaitu opsi yang akan mengakhiri sebuah program
